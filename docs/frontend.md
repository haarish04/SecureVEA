# VEA Fraud Check MVP


### Frontend (React + Vite)
- A minimal modular React application allowing agent to input mobile phone numbers, device details etc and request various potential fraud checks.
- Utilizes Vite for fast dev server startup and easy build steps.

---

## Frontend Setup

### Prerequisites
- Node.js 22+
- npm 

### Installation & Running
```
cd frontend
npm install
npm run dev
```
Frontend runs at `http://localhost:5173`.
